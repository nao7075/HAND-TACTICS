# **ScriptExporter.csの使い方ガイド**

このツールは、プロジェクト内の全C\#スクリプト（Assets以下）の内容を、1つのテキストファイルにまとめて出力します。NotebookLMなどのAIツールにコードを読ませる際に便利です。

## **1\. 導入方法**

1. Unityプロジェクトの Assets/Editor フォルダの中に ScriptExporter.cs を配置します。

## **2\. 使い方**

1. メニューバーから **Tools** \> **Export Scripts for NotebookLM** を選択します。  
2. ファイル保存ダイアログが開くので、保存場所とファイル名を指定して保存します（例: AllScripts.txt）。  
3. 指定した場所にテキストファイルが生成されます。

## **3\. 出力形式**

テキストファイル内では、各ファイルが以下のような区切り線で結合されます。

\--------------------------------------------------  
// Filename: Assets/Scripts/MyScript.cs  
\--------------------------------------------------  
(ここにソースコードの中身)

## **4\. フィルタリングについて**

特定のフォルダ（例: Pluginsなど）を除外したい場合は、ソースコード内の以下のコメントアウトされている部分を有効化してください。

// if (assetPath.Contains("/Plugins/")) continue;
