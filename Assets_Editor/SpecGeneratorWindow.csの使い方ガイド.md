# **SpecGeneratorWindow.csの使い方**

このツールは、プロジェクト内のC\#スクリプトを解析し、クラス構造やドキュメントコメントをまとめたHTML形式の仕様書を自動生成します。

## **1\. 導入方法**

1. Unityプロジェクトの Assets/Editor フォルダ（なければ作成）の中に SpecGeneratorWindow.cs を配置します。  
2. Unityエディタに戻り、コンパイルが完了するのを待ちます。

## **2\. 使い方**

1. Unityエディタ上部のメニューバーから **Tools** \> **Generate Script Specification** を選択します。  
2. 「Spec Gen」というウィンドウが開きます。  
3. ウィンドウ内の **Generate HTML Report** ボタンをクリックします。  
4. 解析が完了すると、自動的に生成されたレポートのフォルダ（ScriptDocs/index.html）が開きます。

## **3\. 生成されるもの**

プロジェクトフォルダ（Assetsの1つ上の階層）に ScriptDocs というフォルダが作成され、以下のファイルが出力されます。

* index.html: 全ファイルの目次  
* styles.css: デザイン用のスタイルシート  
* files/: 各クラスの詳細ページが入ったフォルダ

## **4\. 注意点**

* XMLドキュメントコメント（/// \<summary\>...）をスクリプトに記述しておくと、それらが仕様書の説明文として反映されます。  
* csc.rsp ファイルを併用することで、より正確なコメント取得が可能になる場合があります。
