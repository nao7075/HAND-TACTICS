Class ChatOperationCodePhoton.Chat

Wraps up codes for operations used internally in Photon Chat. You don't have to use them directly usually.

フィールド (Variables)

アクセス名前型説明属性・制約
publicSAuthenticateByte(230) Operation Authenticate.
publicSSubscribeByte(0) Operation to subscribe to chat channels.
publicSUnsubscribeByte(1) Operation to unsubscribe from chat channels.
publicSPublishByte(2) Operation to publish a message in a chat channel.
publicSSendPrivateByte(3) Operation to send a private message to some other user.
publicSChannelHistoryByte(4) Not used yet.
publicSUpdateStatusByte(5) Set your (client's) status.
publicSAddFriendsByte(6) Add friends the list of friends that should update you of their status.
publicSRemoveFriendsByte(7) Remove friends from list of friends that should update you of their status.
publicSSetPropertiesByte(8) Operation to set properties of public chat channel or users in public chat channels.
