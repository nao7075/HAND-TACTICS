Class ChatEventCodePhoton.Chat

Wraps up internally used constants in Photon Chat events. You don't have to use them directly usually.

フィールド (Variables)

アクセス名前型説明属性・制約
publicSChatMessagesByte(0) Event code for messages published in public channels.
publicSUsersByte(1) Not Used. List of users or List of changes for List of users
publicSPrivateMessageByte(2) Event code for messages published in private channels
publicSFriendsListByte(3) Not Used.
publicSStatusUpdateByte(4) Event code for status updates.
publicSSubscribeByte(5) Event code for subscription acks.
publicSUnsubscribeByte(6) Event code for unsubscribe acks.
publicSPropertiesChangedByte(7) Event code for properties update.
publicSUserSubscribedByte(8) Event code for new user subscription to a channel where is enabled.
publicSUserUnsubscribedByte(9) Event code for when user unsubscribes from a channel where is enabled.
publicSErrorInfoByte(10) Event code for when the server sends an error to the client.
