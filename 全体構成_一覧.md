# C#ソースコード構造解析報告書

## 1. 全体構成・一覧

本プロジェクトのソースコードは、主にゲームの中核ロジック（カードシステム、バトル管理、UI制御）、Unityエディタ拡張機能、および外部ライブラリであるPhoton（Realtime/Pun/Chat）のデモ/ユーティリティ群で構成されています。

| 構成カテゴリ | 主な役割 | 関連ファイル数 |
| :--- | :--- | :--- |
| **ゲームコアロジック** | カードデータモデル、コントローラ、UI表示、移動制御、バトル進行管理 | 15 |
| **ゲームシーン管理** | シーン遷移、BGM/SE管理、ホーム/デッキ選択画面制御 | 9 |
| **デッキ/コレクション** | デッキ編集、デッキリスト、マリガンエリアのドロップ処理 | 5 |
| **Unityエディタ拡張** | アセット（画像、構造）のダンプ・収集、スクリプトのエクスポート | 3 |
| **Photon PUN/Realtime** | ネットワーク通信、UIデモ（Cockpit）、ユーティリティ機能 | 多数 (約40+) |
| **アニメーション/ユーティリティ**| DOTween関連、画面アスペクト比制御、演出用スクリプト | 5 |
